# LAB211 

Thực hành lập trình OOP - NangNTH

## Nội dung chính

Repo này chứa các project sau:

- J1.L.P0021: Student Management (175 locs)
- J1.L.P0022: QCandidate Management (175 locs)
- J1.S.P0070: EBank System (150 locs)
- J1.S.P0071: Task Management (150 locs)
- J1.S.P0073: Expense Management (100 locs)

## Cấu trúc thư mục

- J1.L.P0021/: project quản lý sinh viên
- J1.L.P0022/: project quản lý ứng viên
- J1.S.P0070/: project giả lập hệ thống eBank
- J1.S.P0071/: project quản lý task
- J1.S.P0073/: project quản lý chi tiêu

## Môi trường chạy

- Java JDK 8 trở lên
- NetBeans
- Apache Ant 

## Lưu ý

- Mỗi project có cấu trúc theo mẫu MVC cơ bản: model, view, controller, bo, utils
- Một số project có file build.xml và thư mục nbproject để build bằng Ant
